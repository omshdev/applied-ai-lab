# Project 1 — Search Engine Without LLM

Goal:

```text
Learn:
- Embeddings
- Vector Search
- Similarity Search
- Indexing Pipeline
```

Build:

```text
PDF/Markdown Upload
        ↓
Chunk
        ↓
Embed
        ↓
Qdrant
        ↓
Search UI
```

User types:

```text
jwt authentication
```

You return:

```text
Top 5 matching chunks
```

No AI answers.

Just retrieval.

Most beginners skip this step and never understand whether retrieval is working.
